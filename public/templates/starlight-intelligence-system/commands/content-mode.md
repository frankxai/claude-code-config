## Content Frameworks

### AIDA Formula
- **Attention:** Hook that stops the scroll
- **Interest:** Build curiosity
- **Desire:** Create want through benefits
- **Action:** Clear next step

### Value-First Approach
1. Identify audience pain point
2. Provide genuine value/solution
3. Build trust through expertise
4. Soft CTA (not pushy)

### Platform-Specific Best Practices

**LinkedIn:**
- Start with hook (bold statement or question)
- 3-5 key points with examples
- Personal story or insight
- Engagement question at end

**Twitter/X:**
- Thread format for depth
- First tweet = hook + promise
- One idea per tweet
- Visual or example in thread
- CTA in final tweet

**Blog:**
- SEO-optimized headline
- Hook in first paragraph
- Scannable (headers, bullets)
- Actionable takeaways
- Clear CTA

**Email:**
- Subject line (curiosity + benefit)
- Personal opening
- One core message
- Story or example
- Single clear CTA

## Workflow Pattern

### Creating New Content
1. **Define objective:** What's the goal? (educate, inspire, convert)
2. **Know audience:** Who is this for? What do they need?
3. **Choose angle:** What's unique about this perspective?
4. **Draft content:** Write first, edit later
5. **Refine:** Cut 20%, improve clarity
6. **Add visuals:** Graphics, screenshots, formatting
7. **Optimize:** SEO, platform-specific tweaks
8. **Plan distribution:** Where and when to publish
