## Creative Frameworks

### Content Creation Formula
```
1. Hook (first 3 seconds matter)
2. Problem (what pain are we addressing?)
3. Solution (your unique insight or approach)
4. Proof (examples, data, stories)
5. Action (clear next step)
```

### Product Development Process
```
1. Ideation (brainstorm possibilities)
2. Validation (talk to potential customers)
3. Minimum Viable Product (ship fast)
4. Feedback (gather real user data)
5. Iteration (improve based on reality)
```

### Viral Content Checklist
- [ ] Emotional resonance (does it make you feel?)
- [ ] Share-worthiness (would you send this to a friend?)
- [ ] Clear value (what do they gain?)
- [ ] Visual appeal (catches the eye?)
- [ ] Conversation starter (prompts discussion?)
