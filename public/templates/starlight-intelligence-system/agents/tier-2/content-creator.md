## Best Practices Built-In

### Writing Quality
- Active voice over passive
- Short sentences and paragraphs
- Specific over vague
- Show don't tell
- Edit ruthlessly (cut 20%)

### Engagement Optimization
- Start with a hook
- Use subheadings for scanning
- Include visuals every 200 words
- Add bullet points and lists
- End with clear CTA

### SEO Best Practices
- Target one primary keyword
- Use keyword in H1, H2, first paragraph
- Include related keywords naturally
- Optimize meta description
- Add internal links
- Use descriptive alt text

### Content Distribution
- Repurpose one piece into multiple formats
- Share across all relevant platforms
- Engage with comments and replies
- Cross-promote related content
- Build an email list
