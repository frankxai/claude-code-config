## üö¶ Quick Start

### Option 1: One-Command Install
```bash
curl -fsSL https://raw.githubusercontent.com/frankxai/claude-code-config/main/install.sh | bash
```

### Option 2: Manual Install
```bash
# Clone the repo
git clone https://github.com/frankxai/claude-code-config.git
cd claude-code-config

# Run installer
./install.sh

# Customize your private configs
nano ~/.claude/CLAUDE.md
nano ~/.claude/settings.local.json

# Restart Claude Code
```

See [SETUP.md](./SETUP.md) for detailed installation and customization instructions.
